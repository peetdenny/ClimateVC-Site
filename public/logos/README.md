# Portfolio Company Logos

This directory contains placeholder logos for portfolio companies. 

## Replacing Logos

To replace a placeholder with the actual company logo:

1. **Download the official logo** from the company's website, press kit, or brand assets page
2. **Preferred formats:** SVG (best), PNG with transparent background
3. **Recommended size:** Width: 400-800px, maintain aspect ratio
4. **File naming:** Keep the existing filename (e.g., `cur8.png` or `cur8.svg`)

## Where to Find Company Logos

Most companies provide logos in their:
- Press/Media kit (usually linked in footer: "Press" or "Media")
- About page
- Brand guidelines page
- Or contact them directly at press@company.com
